How to work with Git

  1. Create a New Repo on GitHub
  2. Open VSC and in the Get Starter page, click on 'Clone Git Repositories'
  3. Search for the Repo on GitHub and save it locally (better if you save on C disk)
  4. On VSC open a new file index.html inside the repo
  5. Write something and save
  6. Go to Source Control
  7. Click on plus simbol '+' ('Stage Changes')
  8. Give a description to your operations (better if you use the same language for all them)
  9. Click on 'Commit' button, so you had saved the changment, locally
  10. To save it on GitHub, you need to click 'Push' in the menu 'More Actions' (...)
  11. If you have done some mistakes and you want to return to a precedent state of the file, you can use Revert (a soft item)
